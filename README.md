# 🛸 Space Diary

**Space Diary** é um portal digital dedicado à investigação e divulgação de avistamentos de OVNIs no Brasil. O projeto reúne casos históricos, relatos de testemunhas, documentos oficiais e conteúdos informativos sobre fenômenos aéreos não identificados e possíveis contatos extraterrestres.

---

## 🌌 Sobre o Projeto

O objetivo do Space Diary é centralizar informações relevantes sobre ufologia no Brasil, apresentando conteúdos com uma abordagem investigativa e jornalística.

O site aborda desde casos famosos até relatos contemporâneos, oferecendo uma experiência imersiva para curiosos, pesquisadores e entusiastas do tema.

---

## 📂 Conteúdo do Site

O Space Diary inclui:

* 👽 **Casos famosos**

  * Caso Varginha (1996)
  * Operação Prato (1977)
  * Noite Oficial dos OVNIs (1986)

* 📄 **Relatórios oficiais**

  * Documentos da Força Aérea Brasileira
  * Arquivos liberados pelo governo
  * Registros históricos

* 🧑‍🚀 **Depoimentos**

  * Testemunhas civis
  * Relatos de militares
  * Experiências pessoais

## 🚀 Como executar o projeto

3. O site será exibido localmente
